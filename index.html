monitor-input

Switch your Dell U2724DE input from the command line on macOS.

Install

$ curl -fsSL https://rehrumesh.github.io/monitor-input/install.sh | bash

Installs m1ddc via Homebrew if needed, adds the monitor-input command and shell aliases. Apple Silicon + Homebrew required.

Usage

$ monitor-input laptop    # Dell → laptop   (USB-C)
$ monitor-input macmini   # Dell → Mac mini (HDMI)
$ monitor-input toggle    # flip to the other input
$ monitor-input status    # show the current input
$ monitor-input list      # list displays & configured inputs

Aliases: mlaptop, mmini, mtoggle, mstatus. Labels are absolute, so installing on both machines lets you hand the screen back and forth from either keyboard.

How it works

Sends the monitor a DDC/CI command on VCP code 0x60 (input source):

InputVCP 0x60 value
USB-C (laptop)0x19 (25)
HDMI (Mac mini)0x11 (17)

Edit the values at the top of the script to adapt it to other inputs or monitors.
